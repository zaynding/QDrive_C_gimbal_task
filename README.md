# QDrive_C_gimbal_task

基于大疆 C 板（STM32）与 QDrive QD4310 电机驱动的二维云台控制工程。

本项目实现了一个双自由度云台系统，包含 Yaw 与 Pitch 两个方向控制，通过 STM32 控制器完成电机通信、姿态控制以及闭环调节。

---

## 项目简介

本仓库用于记录基于 QDrive 驱动方案的二维云台控制代码。

系统采用 STM32 作为主控制器，通过 CAN 总线与 QD4310 电机进行通信，实现：
- 视觉闭环追踪
- 云台自稳
- 云台电机控制
- 角度闭环控制
- PID 调节
- 电机状态反馈
- 任务调度管理

项目代码结构采用分层设计，便于后续功能扩展。
